# DASACR (DAS28/ACR-EULAR criteria: リウマチ重症度判定)

このiPhone/ iPod touch アプリケーション「DAS28/ACR-EULAR criteria」はDAS28CRP, DAS28ESR, CDAI, SDAI, ACR-EULAR classification criteriaを計算できます。
